Elementos de los formularios

lista de elementos

Nro Nombre de elemento Descripcion Ejemplo
1 input Es un campo de entrada que por medio de su atributo type se puede estipular que tipo de info. requerimos
2 textarea Es un campo de entrada que puede tener multilinea y se difine sus dimensiones mediante las propiedades cols y rows
3 select
  • es un elemento que contiene una lista de opciones a elejir
  • cada opcion esta contenida dentro de otra etiqueta llamada option
4 Button Elemento que representa a un boton, tiene las mismas caracteristicas de qun input:button pero el button tiene etiqueta de apertura y cierre
5 Label es un elemento en el cual se recomienda poner el texto a mostar para que la pagina sea mejor indexada y mediante su propiedad for al hacer click en el label se seleccionara su input
6 fieldset es un elemento en el cual se agrupa otros elementos
Login
7 Legend representa el titulo de algunas etiquetas, por lo general va de la mano con el fieldset

Lista de tipos de input

Numero Tipo Descripcion Ejemplo
1 text Sirve para ingresar texto plano, permite el ingreso de todo tipo de caracteres
2 Email Sirve para ingresar correos electronicos para pesar de ser texto plano, mediante la opcion de validar el formulario se asegura que lo ingresado tenga una @ y un .
3 Number Sirve para ingresar solamente numeros enteros y decimales
4 Date Sirve solo para ingresar fechas, con un calendario de muestra
5 Password Sirve para ingresar las contraseñas, oculta lo ingresado por medio de simbolos
6 Time sirve para ingresar las horas y minutos mas no la fecha
7 Color input para selecionar un color en base a una paleta de colores
8 Checkbox Sirve para selcionar o deseleccionar un grupo de checkboxes, se puede seleccionar uno o varios, van de la mano de su label
9 Radio Sirve para seleccion de un conjunto de opciones, solo se puede seleccionar uno a la vez mediante su propio name
10 File Sirve para subir un archivo local al servidor
11 Reset Limpiar todo el formulario, limpia todos los campos, tiene que ir dentro de una etiquieta Form para que limpie los campos
12 Submit Sirve para enviar todos los campos del formulario, por defecto hara que se recargue la pagina(ejemplo aplicado en la fila 11)
